# 简介

REST API 可以让你用任何支持发送 HTTP 请求的设备来与 DPOpenAPI 进行交互，你可以使用 REST API 做很多事情，比如：

* 一个移动网站可以通过 JavaScript 来获取 DPOpenAPI 上的数据.
* 一个网站可以展示来自 DPOpenAPI 的数据。
* 你可以上传大量的数据，之后可以被一个移动 App 读取。
* 使用任何语言写的程序都可以操作 DPOpenAPI 上的数据。
